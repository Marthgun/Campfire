### CampPlaceableObject/CampPlaceableObjectEx

**SCRIPTNAME.**

    scriptname CampPlaceableObjectEx extends CampPlaceableObject

**OVERVIEW.**

The \`CampPlaceableObjectEx\` script is an expanded version of CampPlaceableObject, for creating more elaborate multi-part placeable objects (such as a barrel with a lantern and a chair). Can be attached to a Furniture object that serves as the main interactible.
